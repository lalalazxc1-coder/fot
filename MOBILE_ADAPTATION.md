# План мобильной адаптации (Responsive Design)

Этот документ содержит полный план работ по адаптации интерфейса системы под мобильные устройства и планшеты. Все изменения отмечаются галочками по мере выполнения.

## 1. Фундамент и Навигация (Shell & Layout)
- [x] **Главный макет (DashboardLayout.tsx)**: Адаптация хедера, скрытие лишних текстов, доработка бургер-меню (Mobile Menu), фикс ширин выпадающих окон (Уведомления, Профиль).
- [x] **Модальные окна (Modal.tsx)**: Запрет выхода модального окна за границы экрана смартфона (100% ширины, отступы, корректный встроенный скролл).
- [x] **Общие компоненты (shared.tsx)**: Адаптация `PageHeader` (перевод кнопок действий под заголовок на узких экранах).

## 2. Основные тяжелые таблицы (Скролл-контейнеры)
Финансовые таблицы и списки со множеством колонок физически не влезают в экран. Делаем для них горизонтальный скролл (`overflow-x-auto`).
- [x] **Сотрудники (EmployeeTable.tsx)**: Добавление контейнера со свайпом, адаптация модалок (Добавление/Редактирование).
- [x] **ФОТ / Планирование (PlanningTable.tsx)**: Добавление враппера со свайпом, возможно липкая левая колонка (sticky) для удобства. Модалки планирования ФОТ переводим в 1 колонку.

## 3. Страницы с карточками и сетками
- [x] **Офферы (JobOffersPage.tsx)**: Сетка карточек `grid-cols-1 md:grid-cols-2 lg:grid-cols-3`. Адаптация огромной 3-колоночной формы создания оффера в вертикальный вид на смартфонах.
- [x] **Заявки (RequestsPage.tsx)**: Сетка заявок и мобильный вид статусов/кнопок.
- [x] **Рынок (MarketPage.tsx)**: Графики, карточки статистики и таблицы.
- [x] **Песочница (ScenariosPage.tsx)**: Адаптация карточек пересчета.
