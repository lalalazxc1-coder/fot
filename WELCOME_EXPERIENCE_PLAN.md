# План реализации модуля "Welcome Experience" 🚀

Этот модуль предназначен для улучшения опыта кандидата в момент принятия оффера. Вместо сухого сообщения об успехе, система показывает интерактивный дашборд с информацией о компании, офисе и первых шагах.

---

## 🏗️ 1. Бэкенд (База данных и API)

### Моделирование данных
*   **Изменение в `JobOfferTemplate`:**
    *   Добавить поле `welcome_content` (тип `JSONB`).
    *   Пример структуры JSON:
        ```json
        {
          "video_url": "https://youtube.com/watch?v=...",
          "office_tour_images": ["url1", "url2"],
          "address": "г. Алматы, пр. Аль-Фараби, 17",
          "coordinates": {"lat": 43.2389, "lng": 76.9455},
          "first_day_instructions": [
            "Прийти к 09:30",
            "Взять оригинал удостоверения личности",
            "Дресс-код: Smart Casual"
          ],
          "merch_info": "Вас ждет Welcome pack: худи, рюкзак и блокнот."
        }
        ```
*   **Обновление схем (Pydantic):**
    *   Добавить поле в `OfferTemplateResponse` и соответствующие модели для создания/обновления.

### Маршрутизация (API)
*   **`GET /api/public/offer/{token}`:**
    *   Обеспечить возврат `welcome_content` в ответе. Можно отдавать его всегда (для превью) или только если статус `Accepted`.
*   **`POST /api/job-offers/{token}/accept`:**
    *   Убедиться, что фронтенд получает обновленный объект оффера с контентом после успешного принятия.

---

## 🎨 2. Фронтенд (Публичная страница)

Работа в `src/pages/requests/PublicOfferPage.tsx`.

### Компонент `WelcomeDashboard`
*   **Эмоциональный блок:** Анимация конфетти (библиотека `canvas-confetti`) при переходе в статус "Принят".
*   **Видео-тур:** Интеграция плеера (YouTube/Vimeo) с обзором офиса.
*   **Информационные карточки (Grid):**
    1.  **"Как нас найти":** Карта (Яндекс/Google) и текстовое описание маршрута/парковки.
    2.  **"Твой первый день":** Инфографика с чек-листом документов и временем встречи.
    3.  **"Твоя команда":** Фото и приветствие будущего руководителя (если данные есть в системе).
    4.  **"Welcome Pack":** Фото или описание корпоративного мерча.

### UX/UI штрихи
*   **Мобильная адаптация:** Один из важнейших пунктов, так как многие смотрят офферы с телефонов.
*   **Кнопка "Добавить в календарь":** Генерация `.ics` файла для первого рабочего дня.

Работа в разделе "Настройки" -> "Шаблоны офферов".

*   **Конструктор контента:**
