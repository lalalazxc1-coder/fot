# Система управления Фондом Оплаты Труда

Система управления Фондом Оплаты Труда (ФОТ), штатным расписанием и сотрудниками.

## Структура проекта

- **backend/**: FastAPI (Python) сервер, база данных SQLite.
- **frontend/**: React (Vite + TypeScript) клиентское приложение.

## Установка и запуск

### 1. Backend (Сервер)

1. Откройте терминал и перейдите в папку `backend`:
   ```bash
   cd backend
   ```

2. (Рекомендуется) Создайте виртуальное окружение:
   ```bash
   python -m venv venv
   ```

3. Активируйте виртуальное окружение:
   - **Windows:** `venv\Scripts\activate`
   - **MacOS/Linux:** `source venv/bin/activate`

4. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

5. Инициализируйте базу данных и создайте Администратора:
   ```bash
   python init_db.py
   ```

6. Запустите сервер:
   ```bash
   python main.py
   ```
   Сервер запустится на `http://127.0.0.1:8000`.

### 2. Frontend (Клиент)

1. Откройте новый терминал и перейдите в папку `frontend`:
   ```bash
   cd frontend
   ```

2. Установите зависимости (если еще не установлены):
   ```bash
   npm install
   ```

3. Запустите режим разработки:
   ```bash
   npm run dev
   ```
   Приложение будет доступно по адресу (обычно) `http://localhost:5173`.

## Доступ в систему

Создается по умолчанию скриптом `init_db.py`:

- **Логин:** `admin`
- **Пароль:** `admin`

## Основной функционал

- **Структура:** Управление филиалами и отделами.
- **План (ФОТ):** Планирование штатных единиц и бюджета.
- **Сотрудники:** Реестр сотрудников, начисление зарплат, синхронизация с планом.
- **Аналитика:** Сравнение плановых и фактических показателей.
- **Пользователи:** Управление доступом и ролями.
